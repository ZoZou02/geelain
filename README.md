# geelain

Geebar快回来...

## 文件结构

- `assets/` - 静态资源目录
  - `favicon/` - 网站图标文件
  - `img/` - 图片资源文件
  - `sound/` - 音频资源文件
- `README.md` - 项目说明文档
- `index.html` - 网站主入口页面
- `style.css` - 页面样式与布局
- `script.js` - 交互动画与逻辑脚本
- `counter_system.js` - 计数器功能模块
- `button_emoji_effect.js` - 按钮表情效果模块
